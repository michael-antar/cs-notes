Performs actions on primitive arrays (like `int[]`, `char[]`). Called with `Arrays.methodName()`.

- `.sort(arr)`: Sorts the array in place.
- `.fill(arr, val)`: Fills the entire array with one value.
- `.toString(arr)`: Get a printable string like `"[1, 2, 3]"`.
- `.equals(arr1, arr2)`: Checks if two arrays have the exact same elements in the exact same order.
