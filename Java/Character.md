# Character Class

Helper methods for manipulating single `char` values. Called using `Character.methodName()`.

- `.isDigit(c)`: Returns `true` if the `char` is a number (`'0'`-`'9'`).
- `.isLetter(c)`: Returns `true` if the `char` is a letter.
- `.isLetterOrDigit(c)`: Returns `true` if the `char` is a letter or digit.
- `.isWhitespace(c)`: Returns `true` if the `char` is a space, tab, etc.
- `.toLowerCase(c)`: Converts a single `char` to lowercase.
- `.toUpperCase(c)`: Converts a single `char` to uppercase.
- `.getNumericValue(c)`: Convert a `char` like `'7'` into the `int` `7`.
