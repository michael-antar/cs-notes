## Collections Class

Utility class that provides static methods for **Collection** interfaces. Called using `Collections.methodName()`.

- `.sort(myList)`: Sorts an `ArrayList` or `LinkedList` in place.
- `.reverse(myList)`: Reverses the order of a `List` in place.
- `.max(myCollection)`: Finds the largest element in any collection (like a `List` or `Set`).
- `.min(myCollection)`: Finds the smallest element.
- `.frequency(myCollection, obj)`: Counts how many times an object appears in a collection.
