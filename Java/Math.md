# Math Class

Static helper methods called with `Math.methodName()`.

- `.max(a, b)`: Returns the larger of two numbers.
- `.min(a, b)`: Returns the smaller of two numbers.
- `.abs(a)`: Returns the absolute value.
- `.ceil(d)`: Rounds a `double` **up** to the nearest integer (returns a `double`).
- `.floor(d)`: Rounds a `double` **down** to the nearest integer (returns a `double`).
- `.round(d)`: Performs a standard "school" rounding (returns a `long`).
- `.pow(a, b)`: Calculates `a` to the power of `b`.
- `.sqrt(a)`: Calculates the square root of `a`.
- `.random()`: Returns a random `double` between `0.0` and `1.0`.
