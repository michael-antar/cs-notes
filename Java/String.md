# String

## Methods

Methods that you call _directly on a string variable_ (e.g., `myStr.methodName()`).

- `.charAt(index)`: Gets the `char` at a specific position.
- `.length()`: Returns the integer length.
- `.substring(begin, end)`: Extracts a new string.
- `.toCharArray()`: Converts the string to a `char[]` so you can use `Arrays.sort()` on it.
- `.split(delimiter)`: Splits a string into a `String[]` array.
- `.toLowerCase()` / `.toUpperCase()`: Returns a new string in the chosen case.
- `.contains(charSequence)`: Returns `true` if the string contains the other string.
- `.isEmpty()`: Returns `true` if the `length()` is `0`.
- `.trim()`: Removes whitespace from the beginning and end.
